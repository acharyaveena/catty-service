## Welcome to Catty Service API Documentation

<!-- vale Google.Headings = YES -->

**Catty** is a comprehensive REST API suite designed to help cat owners and veterinary
professionals make data-driven decisions about feline care. The service consists of two primary
APIs:

- **MeowTrition** - Cat food nutrition and dietary recommendations
- **MeowMetrics** - Cat health and behavior tracking

This documentation focuses on the **MeowTrition API**, which provides detailed nutritional
profiles, ingredient lists, and science-based recommendations for thousands of cat food
products.
